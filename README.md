# UI Kit — HTML & SCSS

Библиотека UI-компонентов, сверстанных на HTML и SCSS.

## Компоненты

- **Кнопки** — primary, secondary, danger, разные размеры
- **Карточки** — с заголовком и текстом
- **Бейджи** — primary, danger, success
- **Алерты** — info, danger, success
- **Формы** — input, textarea, select

## Технологии

- HTML5
- SCSS (с переменными и вложенностью)
- CSS3
