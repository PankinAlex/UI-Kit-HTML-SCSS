# UI Kit — HTML & SCSS

Библиотека переиспользуемых UI-компонентов, созданная на HTML и SCSS с использованием методологии БЭМ.

## Компоненты

- Кнопки: Primary, Secondary, Disabled, размеры small / medium / large, состояния hover / active / focus
- Поля ввода: Normal, Focus, Error, поле с иконкой поиска
- Dropdown: выпадающий список через select
- Карточки: с изображением и без изображения, размеры small / medium / large
- Navbar: адаптивное навигационное меню с гамбургером на мобильных устройствах
- Бейджи и алерты: дополнительные UI-элементы для документации

## Технологии

- HTML5
- SCSS
- CSS3
- Flexbox
- Grid
- БЭМ
- Адаптивная верстка
